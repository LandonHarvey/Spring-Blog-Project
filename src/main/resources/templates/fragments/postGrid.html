Via Quick Trip

Source Code

Third Place: Quick Trip Category. Utilized Vue, Vuex, Vuetify, Node and Express. Application was created to help Via customers and individuals find "quick" trips around San Antonio. Selections Include: Food, Drinks and Attractions. Will locate 20 Attractions within 8 mile radius and allow user to filter by price and category. Allows for user sign in, utilizing passport for google and facebook login. Once logged in will save previous routes and allow user to easily return to trips they have completed.

Reddit 0.5: Ghost Post

Fully functional Reddit Clone. Providing user accounts with Posting, Infinite Nesting Comments, Reporting, Profile Pictures, Upvote and Downvoting of posts. Admin functionality including monitoring reported content (Users, Posts, Comments), Admin levels, Admin creation and tiny purple ghosts.

Vue Budget App

Source Code

A working prototype of a budget application made utilizing Vue.js, Vuex. Allows for budgeting with Categories, Sub-Categories, Transactions. Graphing of Total, Remaining and Spent. with Progress Bars for user experience. Prototype Purchase feature, allowing for searching of products on Ebay using Ebay's Search API. Future will allow price comparison for users to help better manage their finances.
